Manuale

Utente

Come registrarsi?

L'utente ha la possibilità di registrarsi inserendo i dati anagrafici, l'email aziendale e creare una nuova password e visualizzare il messaggio di conferma nell'email.

Come creare un'attività?

Una volta che la registrazione è stata effettuata correttamente si procederà al login, si accede alla pagina home dell'utente. Nella pagina è possibile creare l'attività giornaliera tramite un modulo dove verranno inseriti i dati richiesti.

Come modificare l'attività?

L'utente ha la possibilità di modificare l'attività giornaliera in caso di eventuali errori commessi durante la creazione. L'utente ha anche la possibilità di eliminare l'attività giornaliera creata.

Come visualizzare le attività precedenti?

E' possibile visualizzare le attività precedenti e anche impostare un filtro per ordine di data o attività.

Come reimpostare la password?

L'utente ha la possibilità di cambiare la propria password dalla pagina impostazioni. In caso di password smarrita è possibile richiedere il reset della password.

Attività recenti

L'utente può: modificare, rimuovere l'attività, può cercare per nome o data l'attività con il filtro in alto alla tabella

Impostazioni utente

L'utente può modificare i dati anagrafici e la foto profilo tranne l'email, può cambiare la password.

Admin

Come visualizzare gli Utenti?

L'admin può visualizzare gli utenti nella dashboard facendo il login.

Come accettare gli utenti?

Nella dashboard e nella sidebar è presente il bottone per l'utente d'accettare, con la possibilità di accettare o rifiutare l'utente

Cosa può fare l'admin con l'utente specifico?

-Visualizzare le attività

-Assegnare un attività

-Modificare utente

-Eliminare l'utente

Admin può organizzare la disposizione numerica della tabella per numero di utenti scendendo in basso e inserendo il numero nel filtro

Attività dell'Utente specifico

L'admin può visualizzare le attività dell'utente specifico cliccando sul primo pulsante della dashboard. All'interno delle attività dell'utente l'admin può:

- modificare l'attività

- eliminare l'attività

Admin può anche assegnare una nuova attività, con il bottone in alto alla tabella

Impostazioni admin

L'admin può modificare i dati anagrafici e la foto profilo tranne l'email, può cambiare la password.

Cambio password

Obbligo di compilamento password precedente, nuova password e conferma nuova password
